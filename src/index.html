What is Samantha

Samantha is a tool created to help developers write fluent semantic HTML markup, validate existing code and create smart solutions to improve the markup quality. It is written in vanilla JavaScript and has full browser support

FLUENT HTML

Vanilla JavaScript

Browsers are tolerant to mistakes.

But developers should not be. Perfection in web development seems like something impossible to reach. Samantha.js is the tool that will help you test your markup in an effective, relentless way, in order to allow for perfect semantic HTML, improve search engine optimization and bring up the quality of your code to world standards.
